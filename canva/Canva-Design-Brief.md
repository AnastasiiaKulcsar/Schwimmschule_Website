Use this file in Canva: create a **Brand Kit** (Brand → Brand kit) and add the colors below. Use the same fonts and copy for social posts, flyers, or web graphics that match the website.

## Layout notes (for posters/social)

- **Hero style:** Blue gradient overlay on a water/swimming image; white text; one main headline, one short paragraph, one CTA button.
- **Cards:** White background, light shadow, rounded corners (about 12px), border color `#d0e8ed`.
- **Buttons:** Rounded (8–10px), primary blue or white with blue text; bold/semi-bold text.

## How to use this in Canva

### Option A: All pages as PDF (reference only – text in PDF is not editable in Canva)

1. Open **canva-all-pages.html** (in this **canva** folder) in your browser.
2. Press **Ctrl+P** (or Cmd+P on Mac) → choose **Save as PDF** or **Print to PDF**.
3. Save the PDF. It will have **13 pages** – one per website page.
4. In Canva: **Create a design** → **Uploads** → **Upload** the PDF.
5. Use the PDF as a visual reference only; text inside it is not editable in Canva. For editable elements, use Option A2 below.

### Option A2: Every element editable in Canva (recommended)

- Build the design in Canva with **Text** and **uploaded images** (see **CANVA-EDITABLE-SETUP.md** in this folder).
- Copy all text from **Canva-Content-List.md** into Canva text boxes so headings, paragraphs, and buttons stay editable.
- Upload images from **assets/** and **../images/** (see README in **assets/**).

### Option B: Design board (colors, typography, buttons only)

1. Open **canva-design-board.html** (in this folder) in your browser.
2. Use **Print → Save as PDF** or take a screenshot, then upload that PDF or image to Canva as a reference.

### Brand kit

- Add the hex colors and logo (**assets/banner_swim.png** or **../images/banner_swim.png**) to your Canva **Brand kit**.
- Recreate or adjust layouts using the same colors and fonts in your Canva designs.
