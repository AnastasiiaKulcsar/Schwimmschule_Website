# Alle Website-Seiten als Canva-Projekt hochladen

**→ Schritt-für-Schritt auf Englisch:** **HOW-TO-MAKE-PROJECT-IN-CANVA.md** (in diesem Ordner) – How to make the project in Canva so every element is editable.

---

So laden Sie das komplette Website-Design (alle 13 Seiten) als ein Projekt in Canva hoch und bearbeiten es dort.

---

## Schritt 1: PDF aus der Website-Design-Datei erstellen

1. Öffnen Sie die Datei **canva-all-pages.html** (in diesem **canva**-Ordner) in Ihrem Browser.
   - Doppelklick auf die Datei, oder  
   - Rechtsklick → „Öffnen mit“ → Chrome / Edge / Firefox.
   - **Wichtig:** Der **canva**-Ordner muss im Website-Projekt liegen, damit CSS und Bilder laden (Pfade zeigen auf **../css/** und **../images/**).

2. **Drucken-Dialog öffnen**  
   - **Windows:** `Strg` + `P`  
   - **Mac:** `Cmd` + `P`

3. **Als PDF speichern**  
   - Bei „Drucker“ wählen Sie **„Als PDF speichern“** oder **„Microsoft Print to PDF“** (Windows) bzw. **„Als PDF speichern“** (Mac).  
   - Klicken Sie auf **Speichern** und wählen Sie einen Ordner und Dateinamen (z. B. `Schwimmschule-Delphine-Alle-Seiten.pdf`).

4. Die PDF hat **13 Seiten** – jede Seite entspricht einer Seite Ihrer Website (Startseite, Über uns, Team, Kursangebot, …).

---

## Schritt 2: PDF in Canva als Projekt hochladen

1. Auf **canva.com** einloggen.

2. **Neues Projekt / Design**  
   - Klicken Sie auf **„Erstellen“** oder **„Create a design“**.  
   - Wählen Sie z. B. **„Custom size“** oder ein passendes Format (z. B. Präsentation oder Druck), wenn Sie die Seiten einzeln nutzen wollen.  
   - Oder: Gehen Sie zu **„Hochladen“** / **„Uploads“** (in der linken Leiste).

3. **PDF hochladen**  
   - Klicken Sie auf **„Hochladen“** / **„Upload“**.  
   - Wählen Sie die gespeicherte PDF-Datei (`Schwimmschule-Delphine-Alle-Seiten.pdf`).  
   - Warten Sie, bis der Upload und die Verarbeitung abgeschlossen sind.

4. **Als Projekt nutzen**  
   - Bei mehrseitigen PDFs zeigt Canva oft alle Seiten an.  
   - Sie können die PDF(-Seiten) in ein neues Design ziehen oder als Vorlage nutzen.  
   - So haben Sie **alle Website-Seiten in einem Canva-Projekt** und können Texte, Farben und Layout in Canva bearbeiten.

## Tipps

- **Farben & Logo:** Die genauen Farben und das Logo für Canva stehen in **Canva-Design-Brief.md** (dieser Ordner) (Brand Kit).  
- **Einzelseiten:** Wenn Sie nur eine bestimmte Seite in Canva brauchen: Im Druck-Dialog bei **„Seiten“** nur die gewünschte Seite angeben (z. B. „1“ oder „5–9“), dann als PDF speichern und diese PDF in Canva hochladen.  
- **Datei öffnen:** **canva-all-pages.html** muss aus dem **Website-Projekt** heraus geöffnet werden (am besten aus diesem **canva**-Ordner), damit **../css/** und **../images/** korrekt geladen werden.
