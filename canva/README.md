## Contents

| File / folder | Description |
|---------------|-------------|
| **HOW-TO-MAKE-PROJECT-IN-CANVA.md** | Full guide: Brand Kit, upload images, create 13 pages, add text + images so everything is editable. |
| **CANVA-UPLOAD-ANLEITUNG.md** | Export PDF from HTML and upload to Canva (reference). |
| **CANVA-EDITABLE-SETUP.md** | How to build the design in Canva so every text and image stays editable. |
| **Canva-Design-Brief.md** | Brand colors (hex), fonts, logo, key copy. |
| **Canva-Content-List.md** | All page text to copy into Canva text boxes (German). |
| **canva-all-pages.html** | Open in browser → Print to PDF → 13 pages (one per website page). Use as reference or upload to Canva. |
| **canva-design-board.html** | Color palette, typography, logo, buttons – export as PDF or image for Canva. |
| **assets/** | Logo (`banner_swim.png`) + README listing all images to upload (logo, backgrounds, team photos). |
